# Diwali Sales Analysis Using Python

## Project Overview

This project focuses on analyzing Diwali sales data using Python. The main objective of the project is to perform data cleaning, data preprocessing, exploratory data analysis (EDA), and visualize customer purchasing behavior during the Diwali festival season.

The project helps in understanding:

* Customer demographics
* Purchasing patterns
* Product performance
* Sales trends
* Target customers for business growth

This project was created by learning and implementing concepts from YouTube tutorials along with hands-on practice in Python for Data Analysis.

---

# Key Insights

Some important insights derived from the analysis:

* Married women age group 26–35 years are major buyers.
* Customers from states like Uttar Pradesh, Maharashtra, and Karnataka contribute more to sales.
